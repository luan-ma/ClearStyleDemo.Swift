# 仿Clear手势操作(拖拽、划动、捏合)的UITableView

## 概述

这是一个完全依靠手势的操作ToDoList的演示，功能上左划删除，右划完成任务，拖拽调整顺序，捏合张开插入。

项目源码: https://github.com/luan-ma/ClearStyleDemo.Swift
